# Using-VGGT:Aligning-Point-Clouds-and-Improving-Signle-Image-Inference
Use VGGT to generate point clouds. Align the point clouds to specified coordinate systems that can correspond to real-world geometry. 
Use Qwen-Image_edit-2509 and Loras to generate images of a scene from different angles. Use generated images to improve VGGT inference on the original image.
